# [A+B Problem](https://programming.in.th/tasks/0000)

## คำอธิบายโจทย์
!!! note ""
    โจทย์ให้นำเข้าจำนวนเต็ม $a$ และ $b$ แล้วส่งออกผลรวมของ $a$ และ $b$

## เฉลย
??? note "โค้ด"
    ```c
    #include <stdio.h>

    int main() {
        int a, b;
        scanf("%d %d", &a, &b);
        printf("%d", a+b);
    }
    ```
