## เฉลย

??? note "โค้ดเฉลย"
    ```c
    #include <stdio.h>

    int main() {
        int a, b, gcd;
        scanf("%d %d", &a, &b);
        for (int i = 1; i <= a && i <= b; i++) {
            if (a % i == 0 && b % i == 0) {
                gcd = i;
            }
        }
        printf("%d", gcd);
    }
    ```
